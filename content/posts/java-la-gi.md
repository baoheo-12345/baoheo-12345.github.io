## Java là gì?
Java là ngôn ngữ lập trình hướng đối tượng, đa nền tảng, được phát triển bởi Sun Microsystems (nay thuộc Oracle).

Java cho phép lập trình viên **viết một lần – chạy ở mọi nơi** nhờ cơ chế JVM (Java Virtual Machine).

## Vì sao Java phổ biến?
- Chạy trên nhiều hệ điều hành
- Bảo mật cao
- Cộng đồng lớn
- Dùng nhiều trong backend, Android, doanh nghiệp

## Ví dụ Java đơn giản
```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}
