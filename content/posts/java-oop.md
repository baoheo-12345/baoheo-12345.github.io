## Lập trình hướng đối tượng là gì?
OOP (Object-Oriented Programming) là phương pháp lập trình dựa trên **đối tượng**.

## 4 đặc tính OOP
- Encapsulation (Đóng gói)
- Inheritance (Kế thừa)
- Polymorphism (Đa hình)
- Abstraction (Trừu tượng)

## Ví dụ
```java
class Animal {
    void sound() {
        System.out.println("Animal sound");
    }
}
